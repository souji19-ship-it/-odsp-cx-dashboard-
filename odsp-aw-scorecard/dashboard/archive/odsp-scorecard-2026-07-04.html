Copilot Studio (Dracarys) = Builder platform, excluded from agent counts per the reporting taxonomy. 201/301/401 pulled live via az-cli Kusto from the Dracarys cluster (fdislandsus.centralus + 18 regional peers · DB CAPAnalytics · table TraceEvents · apps CopilotStudio.AgenticRuntime/AgenticLoopApp), 7-day CST windows. ODSP-in-CS = tool calls whose connector is SharePoint/OneDrive (shared_sharepointonline, workiqsharepoint, workiqonedrive, onedriveforbusiness, …); tool metrics are ODSP-specific, turn-success is Copilot-Studio-wide. Volumes (tool calls · tasks · turns) are exact; distinct users/tenants/agents are the sum of per-region distinct counts (≈ global, slight upper bound). Tool-call volumes (201 all-up · 301 ODSP + %-of-CS) are summed across the 19 directly-reachable regional Dracarys clusters — a consistent reachable subset (~64% of the published global total); the global aggregation cluster (fdislandsus.kusto.windows.net) is callout-restricted, so every tool-call figure uses the same reachable basis.
